Kt.it can be accessed via API's too. You can access the API for shortening urls

Shorten a URL

POST /api/shorten

$ curl -X POST http://www.ktit.co/api/shorten -d url['name']=http://bar.com

Response

  {
    "id":9,
    "name":"http://bar.com",
    "code":"ec25",
    "shortened_url":"http://ktit.co/#/ec25"
  }

Details single URL

GET /api/urls/:code

$ curl -X http://www.ktit.co/api/url/ec25

Response

  {
    "id":9,
    "name":"http://bar.com",
    "code":"ec25",
    "shortened_url":"http://ktit.co/#/ec25"
  }

Get all URL's - limited to last 100

GET /api/urls

$ curl -i http://www.ktit.co/api/urls
